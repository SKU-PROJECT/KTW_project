# KickTheWorld
"여행을 통해 새로운 가치를 발견하고 세상에 대한 시야를 넓힌다" <br><br>
- 선호하는 여행취향에 따라 여행일정과 코스를 추천을 하며 인터넷 서칭에 어려운 외국인, 중장년층들을 위한 여행지 추천 서비스를 제공한다.
- 추천 여행지와 여행코스를 선정하여 사용자에게 보여주며 예약 결제까지 한번에 할 수 있도록 한다. <br>
